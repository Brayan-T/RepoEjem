[¿Qué es Git y por qué es importante?] Git es un sistema de control de versiones que nos permite registrar los cambios que hacemos en los archivos de un proyecto, trabajar en equipo y mantener un historial organizado. Fue creado por Linus Torvalds en 2005 y hoy es el estándar en el desarrollo de software. [¿Qué es GitHub?] GitHub es una plataforma que nos permite subir y trabajar con proyectos Git en la nube, facilitando la colaboración y el respaldo remoto. [¿Qué es HTML?] HTML es un lenguaje de marcado que se utiliza para crear la estructura de páginas web. Es fundamental en la creación de cualquier sitio web.
